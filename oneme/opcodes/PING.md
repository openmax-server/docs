# PING (1)
Данный опкод отправляется клиентом раз в 30 секунд клиентом, нагрузка содержит единственный парамтер - "interactive": true.

Сервер отвечаем тем же опкодом, но с пустой нагрузкой.

Пример запроса:
```json
{
  "ver": 11,
  "cmd": 0,
  "seq": 2,
  "opcode": 1,
  "payload": {
    "interactive": true
  }
}
```

Пример ответа:
```json
{
  "ver": 11,
  "cmd": 1,
  "seq": 2,
  "opcode": 1,
  "payload": null
}
```
