# NOTIF_TYPING (129)
Сервер использует этот опкод для оповещения клиента о том, что собеседник печатает.

Пример:
```json
{
  "ver": 10,
  "cmd": 0,
  "seq": 4,
  "opcode": 129,
  "payload": {
    "chatId": 1,
    "userId": 10000
  }
}
```

где:
- chatId - айди чата, в котором собеседник печатает
- userId - айди собеседника
