# CHAT_INFO (48)
Клиент использует данный опкод для запроса информации о чатах.

Пример запроса:
```json
{
  "ver": 10,
  "cmd": 0,
  "seq": 163,
  "opcode": 48,
  "payload": {
    "chatIds": [
      -1
    ]
  }
}
```

где:
- chatIds - массив айди чатов, о которых запрашивается информация
